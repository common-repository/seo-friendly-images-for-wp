This plugin automagically insert/override all the image ALT text to increase SEO image search ranking.

Features of the plugins

== Installation ==

1. Upload 'seo-friendly-images-for-wp' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
